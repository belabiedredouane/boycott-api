# Boycott API

Backend API providing data about alternatives to non-ethical products.

## Installation

```bash
npm install
npm start
```

Server runs on `http://localhost:3000` (or `PORT` env variable).

## API Documentation
- Production: [https://boycott.api.yaqiin.org/api-docs](https://boycott.api.yaqiin.org/api-docs) (Swagger UI)
- Local: [http://localhost:3000/api-docs](http://localhost:3000/api-docs) (Swagger UI)
